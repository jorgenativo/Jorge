Ejercicios de JavaScript (Evaluación Contínua)

1 Operacions bàsiques, Tipus de dades, Prompt i Alert

Crear un programa on l'usuari ingressi en un prompt la temperatura a Celcius i el programa mostri amb un **alert** la temperatura a Fahrenheit.

Demanar un número a un usuari a través d'un prompt i després dividir-lo per 10, exemple: 5/10 igual 0.5

2 If i else

- Crear un programa que determini si un número introduït en un **popup** és divisible per 11 i 5 o no, mostrar el resultat amb **console.log** - Crear un programa que demani a l'usuari dos números en un Prompt i després mostri en un **alert** el nombre més gran.

3 While

Mostrar tots els números de 1 a **N** augmentant de 2 en 2 on **n** ho ingressa l'usuari des d'un prompt.

5-Array

Donat l'array = [1,2,3,4,5,6] - Iterar per tots els elements dins un array utilitzant `while` i mostrar-los a la pantalla. - Iterar per tots els elements dins un array utilitzant `for` i mostrar-los a la pantalla. - Iterar per tots els elements dins un array utilitzant .`for cada element` i mostrar-los a la pantalla. - Donat un array que conté `["blau", "groc", "vermell", "verd", "cafè", "rosa"]` determinar si un color introduït per l'usuari a través d'un prompt es troba dins de l'array o no.

6 Unió, intersecció i compteig

- Hi ha dos arrays, cadascun amb 5 paraules, generar un nou array amb la intersecció dels dos elements. (Exemple: [1,2,3] unió [1,2,4] = [1,2])

- Hi ha dos arrays, cadascun amb 5 paraules, generar un nou array amb la unió d'ambdós elements, (Exemple: [1,2,3] unió [1,2,4] = [1,1,2,2,3,4])
